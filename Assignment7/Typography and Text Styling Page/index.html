Font Families

  • Serif — for formal text
  • Sans-serif — for UI and headings
  • Monospace — for code/data

Tips

  • Use 1.4–1.8 line-height for body copy
  • Use subtle letter-spacing for headings
  • Keep contrast high for legibility
