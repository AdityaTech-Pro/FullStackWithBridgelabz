Core Principles of Responsive Design

Hover over each box to explore the concepts

Fluid Layouts

Layouts that use relative units (%, vw, vh) instead of fixed pixels, allowing content to naturally adapt to different screen sizes. 

.container {
    max-width: 90%;
    margin: 0 auto;
}

.column {
    width: 33.333%;
    padding: 1rem;
}

Flexible Media

Images, videos, and other media that scale within their containing elements while maintaining aspect ratios. 

img {
    max-width: 100%;
    height: auto;
}

.video-wrapper {
    position: relative;
    padding-bottom: 56.25%;
}

Breakpoints

Strategic points where your site's content will respond and adapt to provide the best user experience. 

@media screen and (max-width: 768px) {
    .column {
        width: 100%;
    }
}

@media (min-width: 992px) {
    .nav {
        display: flex;
    }
}
