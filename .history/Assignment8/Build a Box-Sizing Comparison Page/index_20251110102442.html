Box-Sizing Comparison

Compare how different box-sizing values affect the total width of elements with the same CSS properties.

content-box (Default)

Box Properties:

  • width: 300px
  • padding: 20px
  • border: 5px

Total Width: 350px

(width + padding + border)

With box-sizing: content-box, the width property only sets the content width. Padding and border are added to the total width.

border-box

Box Properties:

  • width: 300px
  • padding: 20px
  • border: 5px

Total Width: 300px

(padding and border included)

With box-sizing: border-box, the width property sets the final width. Padding and border are subtracted from the width.

Key Differences

  • Content-box: Total width = width + padding + border
  • Border-box: Total width = width (padding and border included within)
  • Border-box makes it easier to predict final element dimensions
  • Most modern developers prefer border-box for predictable layouts
